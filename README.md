# 課題　 じゃんけんアプリ

## ①課題内容
- 少しだけ変わったじゃんけんをするアプリです。
## ②工夫した点・こだわった点
- ただのじゃんけんだと少し面白くないと思ったので、スタートボタンを押すと、じゃんけんが始まり「ポン!」の合図が出てからすぐ自分の手を出すと勝率が高くなるようになっています。
　（具体的には0.2秒以内に反応できれば100%勝てます。)
- 自分の反応時間はタイマーで表示されるようになっています。
- アニメーションとBGMを付けて少し格闘ゲーム風にアレンジしました。
